README:
Задача:
1.	Создать небольшую страницу «доску объявлений» с минимально необходимым функционалом.
2.	Разработать запросы к РНР файлам в приложении на технологии AJAX.
3.	Минимизировать выборку в SQL запросах к серверу.
4.	Избавиться от перезагрузки страницы при выводе объявлений, постраничной навигации и сортировки.
Что реализовано:
1.	Вывод объявлений, сортировка и постраничная навигация реализована на технологии AJAX.
2.	Минимизирована выборка из БД (происходит запрос, получаем объект и работаем с данными на компьютере пользователя).
3.	Реализован интерфейс для настольных ПК, планшетов (с функцией регистрации нового пользователя и управления страницей пользователя – полный функционал) и мобильных телефонов (без функции регистрации нового пользователя и управления страницей пользователя – только просмотр “доски объявлений”).
4.	Проверка формы регистрации на ввод регистрационных данных (отсутствие их, недопустимость записи и длина записи, функция генерации пароля, шифрование пароля) и входа в аккаунт пользователя (с просмотром вводимого пароля).
5.	Аккаунт пользователя:
- редактирование личных данных пользователя
- добавление нового объявления с картинкой
- просмотр и удаление объявлений пользователя
Что не реализовано:
1.	Проверка полей регистрации нового объявлений.
2.	Кнопка “купить” не работает (нет необходимости, т.к. страничка ознакомительная).
3.	Вывод некоторой информации автора объявления для связи с ним (по необходимости).
4.	Кнопка “добавить новое объявление” на главной странице (слева вверху).
5.	Полная защита (реализована частично) и всевозможные проверки.
6.	Работают только 2 раздела - электроника и одежда.
